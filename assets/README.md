🖼️ Ressources — LM‑Scripts

Ce dossier contient toutes les ressources visuelles du projet LM‑Scripts : logos, bannières, captures d’écran et éléments graphiques utilisés dans la documentation et le README principal.

📂 Contenu

- logo.png — Logo principal du projet  
- logo_circle.png — Version circulaire du logo  
- banner_hacker.png — Bannière façon terminal hacker  
- screenshots/ — Captures d’écran et visuels d’exemples  

🎯 Objectif

Centraliser les éléments graphiques pour :

- uniformiser la présentation du projet sur GitHub  
- faciliter la maintenance et la mise à jour des visuels  
- garantir une structure claire et reproductible  

🧩 Bonnes pratiques

- Utiliser des noms de fichiers courts et explicites (logo.png, banner_dark.png, etc.)  
- Éviter les espaces et majuscules dans les noms  
- Conserver les fichiers sources (ex. .xcf, .psd) dans un dossier séparé ou ignoré par Git  

Exemple de .gitignore local :

`

assets/.gitignore
*.xcf
*.psd
*.tmp
`

🧑‍💻 Notes

Les visuels sont conçus pour être libres d’utilisation dans le cadre du projet LM‑Scripts et peuvent être réutilisés pour la documentation, les présentations ou les pages GitHub.
