📘 Documentation — LM‑Scripts

Ce dossier contient toute la documentation du projet LM‑Scripts, un ensemble de scripts destinés à automatiser l’installation, la configuration et la maintenance de Linux Mint.

📂 Contenu

- CONTRIBUTING.md — Guide pour contribuer au projet  
- CHANGELOG.md — Historique des versions et évolutions  
- main.md — Description détaillée du script principal main.sh  
- README.md — Présentation générale du dossier documentation  

🎯 Objectif

Fournir une documentation claire, organisée et accessible pour :

- comprendre la structure du projet  
- utiliser les scripts efficacement  
- faciliter les contributions  
- suivre l’évolution du projet
