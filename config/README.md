⚙️ Configuration — LM‑Scripts

Ce dossier contient les scripts dédiés à la configuration du système et de l’environnement de bureau sous Linux Mint.

📂 Contenu

- config_cinnamon.sh — Ajuste plusieurs paramètres de l’environnement Cinnamon  
- config_gnome.sh (si ajouté) — Configure certains réglages GNOME  
- config_system.sh — Applique des optimisations et réglages système globaux  

🎯 Objectif

Fournir des scripts simples et reproductibles pour :

- personnaliser l’environnement de bureau  
- appliquer des réglages utiles au quotidien  
- optimiser le système après installation  

🛠️ Utilisation

Depuis la racine du projet :

`bash
./main.sh
`

ou directement :

`bash
bash config/config_cinnamon.sh
`

🧑‍💻 Notes

Ces scripts sont conçus pour être modulaires, sûrs et faciles à adapter selon tes préférences.
