🧰 Utilitaires — LM‑Scripts

Ce dossier contient les scripts utilitaires du projet LM‑Scripts, destinés à fournir des outils pratiques pour diagnostiquer, analyser ou tester rapidement un système Linux Mint.

📂 Contenu

- check_internet.sh — Vérifie la connectivité réseau  
- system_info.sh — Affiche les informations système essentielles  
- benchmark.sh — Réalise un petit benchmark CPU et disque  

🎯 Objectif

Proposer des outils simples et rapides pour :

- diagnostiquer un problème réseau  
- obtenir un aperçu du système  
- tester les performances de base  

Ces scripts sont pensés pour être légers, fiables et faciles à utiliser.

🛠️ Utilisation

Depuis la racine du projet :

`bash
./main.sh
`

Ou directement :

`bash
bash utils/check_internet.sh
`

🧑‍💻 Notes

Les scripts utilitaires sont conçus pour être indépendants, modulaires et adaptables selon tes besoins.
